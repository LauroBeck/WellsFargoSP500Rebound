## 🏗 Architecture Overview
This repository implements a modular C++ engine designed to track and simulate market inflection points within the Nasdaq ecosystem, with a specific focus on **IBM** and **Oracle (ORCL)**.

### 🧩 Core Modules
* **`engine/InflectionEngine`**: Real-time volatility and breakout detection (Current Threshold: 5.0%).
* **`engine/Sub1nmTracker`**: Yield simulation for High-NA EUV lithography and Nanostack architectures.
* **`utils/EarningsParser`**: Analytical logic for comparing reported vs. consensus EPS with surprise delta calculations.

## 🚀 Getting Started

### Prerequisites
* **Compiler**: GCC/G++ 9.0+ or Clang 10.0+
* **Build System**: CMake 3.15+

### Installation
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/IBMOracleInNasdaq.git
cd IBMOracleInNasdaq

# Build and Run
mkdir build && cd build
cmake ..
make
./MarketMonitor
```

### Testing
To verify the fundamental analysis logic and surprise calculations:
```bash
g++ -std=c++17 tests/test_main.cpp -o run_tests && ./run_tests
```
